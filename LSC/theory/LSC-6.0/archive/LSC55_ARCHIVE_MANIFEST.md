# LSC 5.5 Archive Manifest

Compilation date: 2026-04-25

## Archive purpose

This archive preserves one coherent publication-oriented working version of the
LSC framework together with core sources, historical context, and simulation
support material.

  Main statement of gravitationally coupled neutrino evolution.

  Expanded version of the relativistic energy-shift argument.

  Intermediate scalar-parameter version using `delta`.

  Transition from scalar `delta` to tensor `D_{mu nu}`.

  Bridge between LSC 4.2 and LSC 5.0@.

  Quantum-dynamical extension.

  Final synthesis and consistency notes.

## Files deliberately excluded

- `Kuendigung_Arbeitsverhaeltnis.pdf`
  Private personal document unrelated to the theory.
- `GeRysy.pdf`
  Unrelated scientific document outside the LSC framework.
- `OBS Studio - Screen Recorder_5.0_APKPure.apk`
  Binary file unrelated to the scientific content.
- `*.gif`, `*.mp4`
  Visual presentation assets not required for the publication archive.
- `LSC_QUE.pdf`, `LSC_QUE_Report.pdf`
  The quantum/decoherence branch remains too weakly constrained to serve as the
  mainline of the final theory.
- duplicate files such as `ZENODO_LSC_4_2_ULTRA_FULL.pdf`, `LSC_REZ.pdf`,
  `LSC_REZ-1.pdf`
  Excluded where the content is redundant with the selected canonical version.
- legacy `*.zip` packages
  Not duplicated inside the new final archive.

## Selection criteria

1. Priority was given to documents that reinforce the coherent progression
   `LSC 4.2 -> LSC 5.0@ -> LSC 5.5`.
2. Mathematical and theoretical content was preferred over presentation assets.
3. Documents that weakened internal consistency or remained too speculative were
   not placed in the core package.
4. The archive is intended as a preprint-working package rather than a raw dump
   of the full directory.
