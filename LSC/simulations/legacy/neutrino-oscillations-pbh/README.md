## Kompilacja dokumentu
1. Upewnij się, że masz zainstalowany system LaTeX (np. TeX Live) oraz biblatex z biberem.
2. W katalogu głównym uruchom:
```bash
pdflatex main.tex
biber main
pdflatex main.tex
pdflatex main.tex
```
