# External Review 2: Errata - Measurement Framework Interpretation

Date: 2026-04-26

## Author's Note

The Gemini measurement-framework report is preserved for transparency, but its framing should not be treated as the primary interpretation of LSC 6.0.

## Correction of Record

**Incorrect interpretation**: LSC 6.0 is a new measurement method for scintillation detectors or a detector-metrology hardware project.

**Correct interpretation**: LSC 6.0 is a theoretical neutrino phenomenology model. It uses a detector-response tensor as part of the mathematical description, but it does not propose R&D on PMTs, scintillators, electronics, or detector hardware.

## Why This Matters

LSC 6.0 requests computational support for theory development, tensor calculations, scans, and possible future public-data studies. It should not be presented as a detector-construction or scintillation-measurement project.
