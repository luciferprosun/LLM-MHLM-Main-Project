# Project Summary for Grant or Hardware Support

## Short Summary

LSC 6.0 is an independent phenomenological research project in neutrino physics. It proposes a testable framework linking weak propagation-level effects with anisotropic detector response. The immediate goal is to determine whether Gallium source deficits can be modeled without requiring a large universal energy-scale distortion or an unconstrained sterile-neutrino interpretation.

## Why Compute and LLM Support Matter

The next stage requires:

- converting the phenomenological ansatz into reproducible numerical models,
- fitting Gallium source data with real line energies and cross sections,
- adding KATRIN and IceCube constraints,
- scanning tensor-response parameter space,
- generating arXiv-ready figures, tables, and uncertainty analyses,
- using LLM-assisted literature review and symbolic checking to accelerate manuscript preparation.

## Requested Direction

The project would benefit from access to:

- a reliable PC or workstation,
- GPU or cloud compute for parameter scans,
- LLM access for literature organization, code generation, and manuscript polishing,
- collaboration with physicists familiar with neutrino phenomenology and detector systematics.
