# Data Notes

This dataset is a compact public-summary dataset for toy validation only.

The BEST inner/outer ratios were checked against the NIST publication page for
Barinov et al., Phys. Rev. Lett. 128, 232501 (2022). The GALLEX reanalysis
values were checked against the ScienceDirect snippet for the 2010 reanalysis.
The SAGE and BEST extraction-level rows are transcribed from the user-provided
Gemini Deep Research report and should be manually checked against the cited
papers before being used for a strong statistical claim.

No official covariance matrix is included. The validation script therefore uses
diagonal uncertainties and simple toy templates. Its output is suitable for
methodological illustration, not for a collaboration-grade likelihood
reproduction.
