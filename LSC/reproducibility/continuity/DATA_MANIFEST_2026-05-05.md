# Data Manifest

## `data/codex data.pdf`

PDF report generated from the Codex data package for BEST, GALLEX and SAGE modeling.

## `data/dataset.json`

Structured dataset containing experiment names, detector geometry, source properties, measured results, cross-section entries, references and extraction status.

## `data/summary_report.md`

Human-readable summary of what is complete, missing or still needs verification.

## `data/loader.py`

Simple Python loader with functions such as `load_dataset()` and `get_experiment("BEST")`.

## `research_plan/LSC_BEST2_MHLM_RESEARCH_PLAN.md`

Working plan for the dual-pole research structure: LSC/BEST-2 physics development plus MHLM/AI-research documentation.

## `mhlm_ai_research/`

Meta-research wing for tracking how multiple LLM systems contribute to theory development, including hallucination and overclaim analysis.

## `mhlm_ai_research/gemini/GEMINI_LSC_WORKING_REPORT_2026-04-30.md`

Provisional Gemini report summary based on the user-provided draft. This will be replaced or expanded when the final Gemini report is provided.

## `session_notes/2026-04-30_SESSION_STATE.md`

Session state for continuation: repo links, selected theory base, current data status and guardrails.

## `reports/github_traffic/2026-04-30_github_traffic_report.md`

Day-zero GitHub traffic report for the public repository. It records views, clones, stars, forks, issues and the current limits of what GitHub can prove about outside testing.
