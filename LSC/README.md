# LSC Research Track

LSC is the neutrino-phenomenology branch focused on gallium anomaly interpretation and falsifiable detector/propagation modeling.

## Focus

- BEST, GALLEX, SAGE anomaly context
- detector-response anisotropy and propagation coupling
- reproducible simulation and fit pipelines
- explicit external constraint checks (KATRIN, IceCube)

## Validation Roadmap

- frozen prediction protocols before fitting
- leave-one-out cross-checks
- cross-experiment transfer tests
- explicit null-model and sterile-neutrino benchmark comparisons
