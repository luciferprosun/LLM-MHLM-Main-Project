## Interpretation

The preferred repository framing is:

> LSC 6.0 is a phenomenological neutrino model proposal combining propagation-level effects with anisotropic detector response.

The measurement-framework PDF is retained with errata because it can be misread as describing detector hardware or scintillation metrology. That is not the intended scope of LSC 6.0.
